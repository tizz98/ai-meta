|---|---|

| `./meta wave` | Plan a parallelizable wave of sub-issues |

| `./meta check` | Enforce codified standards (CI runs `--strict`) |

| `./meta stats` | Repo analytics: commit counts by author, lines of code by language |

| `./meta tag` | Cut a release (version bump + tag + push) |
| `./meta upgrade` | Pull framework updates + migrate meta.toml |

## Conventions

- Task state is carried by `status:*` labels (todo/in-progress/blocked/done).
- Standards are codified in `.meta/meta.toml` and enforced by `check` (and CI).
- `task`/`milestone`/`setup` need a GitHub token (`GITHUB_TOKEN`, or `gh auth`).
