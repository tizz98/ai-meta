# meta upgrade

GENERATED by ai-meta (`meta init`/`meta upgrade`). Do not edit; change
`.meta/meta.toml` and run `./meta upgrade`.

Regenerate managed artifacts and migrate meta.toml to a newer ai-meta.

## Usage

```
./meta upgrade [--dry-run]
```

Run `./meta upgrade --help` for the full option list.
